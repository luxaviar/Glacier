# Glacier

A personal toy game engine project based on D3D11 & D3D12

## Prerequistes
* Visual Studio 2019
* Windows 10

## Features

* D3D11 & D3D12 backend
* PBR & IBL
* HDR/linear lighting
* Cascaded Shadow Map
* MSAA
* Rigidbody Physics
* Component-Based Architecture
* Editor & Gizmo
* Input System
