# Glacier

A personal toy game engine project based on D3D11

## Prerequistes
* Visual Studio 2019
* Windows 10

## Features

* PBR & IBL
* HDR/linear lighting
* Cascaded Shadow Map
* Rigidbody Physics
* Component-Based Architecture
* Editor & Gizmo
* Input System
